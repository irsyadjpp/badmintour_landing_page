# **App Name**: BadminTour Landing

## Core Features:

- Hero Section: Dynamic hero section featuring a badminton player in action and bold typography to capture attention.
- Service Display: Bento Grid layout showcasing the core services: Mabar, Sparring, and Coaching, each represented with rounded cards.
- Floating Navigation: Implement a floating navigation bar with a glassmorphism effect for a modern, Gen-Z aesthetic.
- Community Avatars: Display profile avatars of community members to foster a sense of belonging and inclusivity.
- Event Cards: Design event cards that clearly show the date, time, and a progress bar for slots remaining, encouraging quick sign-ups.

## Style Guidelines:

- Primary color: Vibrant red (#CA1F3D) to convey energy and excitement.
- Accent color: Bright yellow (#FFBE00) for CTAs and highlights to draw attention.
- Background color: Light gray (#F0F0F0) provides a clean, modern backdrop.
- Body and headline font: Use geometric sans-serif fonts to complement the design with a modern, sporty, and energetic feel. Note: currently only Google Fonts are supported.
- Implement a mobile-responsive layout to ensure the website looks great on all devices.
- Use ample white space to enhance readability and maintain a clean, professional look.
- Utilize icons that represent the different services and community aspects, creating a consistent and visually appealing experience.
- Subtle animations and transitions to create a dynamic and engaging user experience.
